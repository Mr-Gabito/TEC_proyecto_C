# **Tienda de libros** 

## Problema
Te han contratado para hacer una plataforma sencilla en linea que puede facilitar el manejo del inventario en la tiende de libros. Tendras que incluir los libros que estan a la venta junto la información necesaria de los libros. La plataforma tiene que ser accedida por el empleado para que pueda establecer el inventario, y ver los precios de cada libro.

### Libro
Se tendra que crear una clase del libro que tenga la siguiente información: título, ID del libro, género, número de hojas, y el precio. Tendras que incluir tres clases más para los diferentes tipos de libros siendo: libro de pasta blanda, libro de pasta dura, y libro digital.

### Libro de pasta blanda
Para esta clase, sera para especificar los libros, que no tendrian ni un metodo o atributo extra, ya que seria el libro base.

### Libro de pasta dura
Para esta clase, se tendra que hacer un metodo donde cambia le precio del libro de pasta dura si es una versión especial, y tambien menciona que se incluye en la version especial.

### Libro digital
Para esta clase, el precio cambiaria ya que es digital y tambien tendra un metodo que te avisa si tiene audio libro o no, lo cual viene incluido en la compra.

## Consideraciones
Este proyecto esta hecho en C++ y solo puede correr como .exe, se tendra que hacer esta compilacion con g++.
